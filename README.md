\# Invoice and Billing System (C++)



A simple \*\*menu-driven console application\*\* built using \*\*C++\*\*.



\## Features

\- Add and delete users

\- Select items and generate bills

\- Calculates GST and discount

\- Generates invoice with date \& time

\- Saves invoice history to file



\## How to Run

```bash

g++ main.cpp billing.cpp -o billing

./billing
